# AGENTS.md

## Workflow Rules
- Before starting any new PR-sized task, confirm you are in the correct project folder.
- Switch to local `main` before starting new PR work.
- Update local `main` from `origin/main`.
- Create the new working branch from that updated `main`.
- Do not start new PR work from a stale branch.

## Documentation Source of Truth
- Before starting product work, read `docs/spec.md`.
- Use `docs/pr-plan.md` to keep the work aligned to the intended PR slice.
